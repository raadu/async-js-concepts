XMLHttpRequest

XMLHttpRequest (also called XHR) is a web API (object) that is used to retrieve data through http requests.

XMLHttpRequest.readyState States

  • 0 = UNSENT, client created, open not called yet
  • 1 = open() called
  • 2 = send() called
  • 3 = downloading response
  • 4 - response download completed
